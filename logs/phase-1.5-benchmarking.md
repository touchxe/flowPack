# [Phase 1.5] 작업 로그 — 벤치마킹

## 기본 정보
- **날짜**: 2026-03-31
- **Phase**: Phase 1.5: 벤치마킹
- **소요 시간**: 약 1시간
- **관련 Task**: PHASE-1.5

## 이번에 한 일
FlowPack의 주요 경쟁사인 Mirra(mirra.my)를 공개 랜딩페이지와 요금제 페이지를 통해 분석했다. 기능, 요금제, UX 패턴, 채널 지원 현황을 FlowPack 계획과 대조하여 차별화 포인트를 도출했다. 사이트맵도 벤치마킹 결과를 반영해 확정했다.

## 핵심 결정 사항
| 결정 | 내용 | 근거 |
|------|------|------|
| 차별화 1 | 네이버 블로그 연동 유지 | Mirra 미지원, 한국 시장 핵심 |
| 차별화 2 | 카카오톡 알림 유지 | Mirra 미지원, 국내 사용자 친화 |
| 차별화 3 | `/ai/bulk-generate` 유지 | Mirra 미지원 |
| 채택 1 | 7일 무료 체험 (카드 불필요) | Mirra에서 검증된 온보딩 패턴 |
| 채택 2 | 크레딧 기반 과금 검토 | Phase 3 아키텍처에서 결정 |
| 2차 개발 이관 | Threads/TikTok/YouTube, 댓글·DM 관리, 팀 워크스페이스 | MVP 범위 초과 |

## 사용한 도구/기술

## 어려웠던 점 / 배운 점
- 로그인 이후 내부 UX는 확인 불가 (공개 정보만 분석) → 실제 사용 경험은 무료 체험 계정 직접 생성 권장
- Mirra 요금제는 달러 기준. FlowPack의 한국 원화 가격 정책은 Phase 3 아키텍처 확정 시 결정 필요

## AI와의 협업 노트

- 기능 대조표 자동 생성 후 차별화 포인트 명확화
