### 2.2 AI 페르소나 (차별화 포인트)

Mirra의 핵심 차별화는 **AI 페르소나 기술**:
- 과거 게시물 분석 → 브랜드 고유 언어/스타일 자동 학습
- 페르소나 카피 (경쟁 계정 스타일 모방)
- 페르소나 직접 설계 (AI와 대화로 브랜드 컨셉 설계)
- 페르소나 템플릿 (전문가 스타일 즉시 적용)

→ FlowPack의 **"내 캐릭터/글 작성 룰 설정"** 기능과 방향 일치. Mirra가 더 세분화되어 있음.

### 4.2 주요 페이지별 UX 패턴

#### 홈 (대시보드)
- 환영 메시지 + 퀘스트 0/3 진행바 + 크레딧 표시 (60 크레딧)
- 핵심 기능 3개 퀵 카드 (카드뉴스, 숏폼 영상, AI 콘텐츠 아이디어)
- 각 카드에 보너스 크레딧 인센티브 표시 (+15 크레딧 & +12h)
- 온보딩 게이미피케이션: 첫 사용 무료 + 크레딧 보상

#### 카드뉴스 생성 (/carousel-lab)
- 3단계 마법사: `디자인 선택 → 콘텐츠 입력 → 결과 확인`
- 시작 방법 3가지 선택: "바로 만들기" / "템플릿 둘러보기" / "내 디자인으로 만들기"
- 탭: 카드뉴스 생성 | 나만의 디자인 학습 | 카드뉴스 생성기록

#### 블로그 생성 (/ai/longform)
- 탭: 제작 | 글쓰기 스타일
- 상태 필터: 전체 / 초안 / 완료 / 보관됨
- 빈 상태: 일러스트 + 설명 + CTA 버튼

#### SNS 계정 관리 (/social-accounts)
- 빈 상태: "연동된 SNS 계정이 없습니다"
- 연동 후 혜택 4가지 미리 보여줌 (AI 맞춤 콘텐츠 제안, 콘텐츠 예약, 성과 분석, 댓글/DM 자동 응답)
- 강한 CTA 유도

#### 콘텐츠 캘린더 (/calendar)
- 월/리스트 뷰 토글, 주/월 단위 전환
- SNS 미연동 시 보라색 잠금 모달 오버레이 → 계정 연동 유도 게이팅

#### 대량 기획 (/ai/bulk-generate)
- 3단계 마법사: AI 기반 아이디어 → 선택 → 생성
- SNS 계정 연결 후 오디언스 분석 기반 맞춤 아이디어 생성

### 4.3 디자인 시스템 관찰 (⚠️ AI 슬롭 경고)
- **Primary 색상**: 보라색/바이올렛 (#7C3AED 계열) — **전형적 AI 제품 색상**
- **Evaluator 기준 위험**: Mirra의 보라 그라데이션 + 보라 CTA 버튼은 "독창성" 기준 FAIL 가능성
- **FlowPack 권고**: 보라색 계열 사용 금지. 차별화된 색상 팔레트 필수 (Phase 2.5에서 확정)

### 4.4 온보딩 패턴
- 무료 7일 체험 → 상단 고정 D-7 카운트다운 배너
- 퀘스트 시스템: 핵심 기능 첫 사용 시 크레딧 + 시간 연장 보상
- 빈 상태에서 항상 기능 혜택 미리 제시 후 CTA

### 4.5 소셜 로그인
- Mirra: Google + Kakao (Apple 없음)
- FlowPack: Google + Kakao + Apple (계획)

## 8. 결론

FlowPack은 Mirra와 유사한 포지션이나 **한국 시장 특화** (네이버 블로그, 카카오톡 알림, 완전 한국어 UX)로 차별화 가능. Mirra의 요금 구조와 온보딩 패턴은 FlowPack에도 적용 권장. MVP 범위 내에서 `/ai/bulk-generate` (대량 기획)는 Mirra에 없는 명확한 차별점.
