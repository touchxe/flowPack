## shadcn 컴포넌트 목록
| 컴포넌트 | 용도 |
|---------|------|
| Card | 통계 카드, 빠른 시작 카드, SNS 안내 |
| Table | 최근 콘텐츠 목록 |
| Badge | 콘텐츠 유형, 상태 표시 |
| Progress | 무료 플랜 생성 수 진행바 |
| Button | 시작하기, SNS 연동 |
| Avatar | 사용자 프로필 |
| DropdownMenu | 콘텐츠 액션 (편집/삭제/배포) |
| Alert | SNS 미연동 안내 |

- 전체: `flex h-screen`
- 사이드바: `w-60 border-r`
- 메인: `flex-1 overflow-auto p-6`
- 통계 카드: `grid grid-cols-3 gap-4`
- 빠른 시작: `grid grid-cols-2 gap-4`
