## shadcn 컴포넌트 목록
| 컴포넌트 | 용도 |
|---------|------|
| Tabs | 생성/학습/기록 탭, 편집 패널 탭 |
| Card | 방법 선택 카드 |
| Textarea | 주제 입력, 슬라이드 텍스트 편집 |
| Select | 톤/스타일/장수 옵션 |
| RadioGroup | 이미지 생성 방식 |
| Input | 참고 URL |
| Checkbox | 배포 채널 선택 |
| Button | 생성, 저장, 배포 |
| Badge | "추천" 태그 |
