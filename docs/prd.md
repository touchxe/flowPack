## 1. 프로젝트 개요

### 1.1 문제 정의
스타트업·개인사업자·소호 사업자는 외부 에이전시나 전문 마케터 없이 SNS/블로그 홍보 콘텐츠를 꾸준히 생산·배포하는 것이 어렵다. 비용과 인력 모두 부족하기 때문이다.

### 1.2 솔루션 요약
AI가 홍보 콘텐츠(카드뉴스·장문 블로그·영상)를 자동 생성하고, Instagram·Facebook·LinkedIn·워드프레스 등 여러 채널에 원클릭 배포할 수 있는 **올인원 홍보 자동화 플랫폼**.

### 1.3 대상 사용자
| 사용자 유형 | 설명 | 핵심 니즈 |
|------------|------|----------|
| 스타트업 | 홍보 예산·인력 부족, 빠른 성장 필요 | 빠르고 저렴한 콘텐츠 생산 |
| 개인사업자 | 1인 운영, 마케팅 전문성 부족 | 전문가 없이도 완성도 있는 홍보 |
| 소호 사업자 | 소규모 팀, 반복 홍보 업무 부담 | 반복 작업 자동화, 다채널 배포 |

### 1.4 성공 지표
| 지표 | 목표값 | 측정 방법 |
|------|--------|----------|
| 월간 홍보글 생성 수 | 1,000건 | DB 집계 |
| MAU (월간 활성 사용자율) | 30% | 로그인 기준 |
| 무료→유료 전환율 | 5% | 결제 전환 |
| NPS | 50점 | 설문 |

## 2. MVP 범위

### 2.1 포함 (In Scope)
- 소셜 로그인 (Google, Kakao, Apple) + 일반 로그인
- AI 홍보글 생성 (카드뉴스·장문 블로그·영상·URL→콘텐츠·대량 기획)
- 직접 편집 에디터 + 저장/수정/삭제
- AI 이미지 생성
- SNS 연동 배포 (Instagram, Facebook, Twitter/X, LinkedIn)
- 블로그 연동 배포 (네이버 블로그, WordPress)
- 콘텐츠 캘린더 관리
- 포스팅 조회수 통계 & 대시보드
- 무료 티어 + 월간/연간 유료 요금제
- 이메일·카카오톡 알림
- 마이페이지 (프로필, 글쓰기 룰/캐릭터 설정)
- 고객 지원 (FAQ, 1:1 문의, 가이드, 사례, 리뷰)

### 2.2 제외 (Out of Scope — 2차 개발)
- Homepage 제작 기능

### 2.3 미확정 항목
| 항목 | 상태 |
|------|------|
| 무료 티어 월간 생성 수 상한 | ⚠️ 확인 필요 |
| 유료 요금제 구체적 금액 | ⚠️ 확인 필요 |
| 소셜 로그인 최종 제공사 확정 | ⚠️ Google·Kakao·Apple 추정 |

## 3. 사용자 스토리 (핵심)

### Epic 1: 인증
- `US-001` 사용자는 Google/Kakao/Apple 소셜 계정으로 가입·로그인할 수 있다
- `US-002` 사용자는 이메일·비밀번호로 가입·로그인할 수 있다
- `US-003` 사용자는 비밀번호를 이메일로 재설정할 수 있다
- `US-004` 동일 계정의 중복 로그인이 방지된다

### Epic 2: 콘텐츠 생성
- `US-010` 사용자는 업종·톤·스타일·길이를 선택해 AI 홍보글을 생성할 수 있다
- `US-011` 사용자는 생성된 글을 직접 편집할 수 있다
- `US-012` 사용자는 AI로 관련 이미지를 생성할 수 있다
- `US-013` 사용자는 URL을 입력해 콘텐츠로 변환할 수 있다
- `US-014` 사용자는 카드뉴스(Carousel) 형식으로 생성할 수 있다
- `US-015` 사용자는 장문 블로그 포스트를 AI로 생성할 수 있다

### Epic 3: 배포
- `US-020` 사용자는 SNS 계정(Instagram·Facebook·Twitter/X·LinkedIn)을 연동할 수 있다
- `US-021` 사용자는 블로그(네이버·WordPress)를 연동할 수 있다
- `US-022` 사용자는 여러 채널에 원클릭으로 동시 배포할 수 있다

### Epic 4: 통계 & 관리
- `US-030` 사용자는 포스팅별 조회수를 확인할 수 있다
- `US-031` 사용자는 콘텐츠 캘린더에서 예약·관리할 수 있다

### Epic 5: 결제
- `US-040` 무료 사용자는 월간 제한 내에서 서비스를 이용할 수 있다
- `US-041` 사용자는 월간/연간 요금제로 업그레이드할 수 있다

## 5. 엣지 케이스

| 케이스 | 처리 방향 |
|--------|---------|
| AI 생성 API 오류·타임아웃 | 재시도 3회 후 사용자에게 안내, 크레딧 미차감 |
| SNS 토큰 만료 | 재인증 유도 알림, 배포 실패 시 명확한 오류 메시지 |
| 무료 티어 생성 수 초과 | 업그레이드 유도 모달, 부드러운 게이팅 |
| 다채널 동시 배포 부분 실패 | 채널별 성공/실패 상태 개별 표시 |
| 이미지/영상 생성 지연 | 비동기 처리 + 완료 알림 (이메일/카카오톡) |
