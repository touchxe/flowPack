### 비회원 페이지 설계 원칙 (Mirr 참고)

1. **Landing Page**: 명확한 가치 제안 + 즉시 action 가능한 CTA
2. **요금제**: 비회원이 가격을 확인할 수 있어 신뢰도 향상
3. **이용약관/개인정보/쿠키**: 법적 필수 항목 (Mirr footer 참고)
4. **문의하기**: 고객 지원으로 전환율 향상

### Mirr 벤치마킹 Insights

- **Header**: Logo, 제품, 기술, 요금제, 제휴 파트너, 문의하기, 로그인/무료 시작
- **Landing 핵심 메시지**: "마케팅, 직접 하지 마세요. 콘텐츠부터 관리까지, AI가 다 도와드립니다."
- **문제점 4가지**: 기획의 고통, 제작 시간 부족, 일관성 실패, 반응 없는 콘텐츠
- **Pain Points**: 8가지 고민 포인트 (복붙 지옥, 알고리즘 스트레스 등)
- **기능**: 카드뉴스, 텍스트 콘텐츠, 블로그, 숏폼 영상 + AI 페르소나
- **비교표**: ChatGPT vs Mirr vs Canva/CapCut
- **고객 후기**: 실제 사용자 리뷰 2건
- **Footer**: 커뮤니티, 비밀번호, 블로그, 이용약관, 개인정보처리방침, 쿠키 정책
