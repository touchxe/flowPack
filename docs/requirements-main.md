# FlowPack — 요구사항 정의서

> **작성일**: 2026-03-31  
> **Phase**: Phase 1 (인터뷰 에이전트 완료)  
> **상태**: 확정 (미확정 항목 별도 표기)

---

## 1. 핵심 문제 정의

외부 에이전시나 전문가 도움 없이, **자급자족 온라인 홍보 활동이 가능한 AI 기반 플랫폼**.

스타트업·개인사업자·소호 사업자가 전문 마케팅 팀 없이도 SNS/블로그 홍보 콘텐츠를 AI로 생성하고, 한 번에 여러 채널에 배포할 수 있도록 한다.

---

## 2. 대상 사용자

| 유형 | 특징 |
|------|------|
| 스타트업 | 홍보 자원 부족, 빠른 성장 필요 |
| 개인사업자 | 비용 절감, 자급자족 마케팅 필요 |
| 소호 사업자 | 전문 마케팅 팀 부재, 간편 홍보 필요 |

## 4. MVP 기능 (1차 개발)

### 4.1 인증 & 계정
- 소셜 로그인: Google, Kakao, Apple
- 일반 회원가입 / 로그인
- 비밀번호 찾기
- 동일 계정 중복 로그인 방지

### 4.2 홍보글 작성
- AI 기반 홍보글 생성
- 직접 작성/편집 에디터
- 저장 / 수정 / 삭제
- 이미지 생성 (AI)
- 글 작성 옵션: 톤, 스타일, 길이

### 4.3 콘텐츠 유형
- 카드뉴스 (Carousel) 생성
- 영상 생성
- 블로그 장문 생성 (AI Longform)
- URL → 콘텐츠 변환
- 대량 기획 및 생성

### 4.4 배포
- SNS 연동: Instagram, Facebook, Twitter/X, LinkedIn
- 블로그 연동: 네이버 블로그, WordPress
- SNS 인증 기반 연결 관리
- 원클릭 배포

### 4.5 블로그
- 홍보용 블로그 템플릿 제공
- 블로그 포스팅 관리

### 4.6 고객 지원
- FAQ
- 1:1 문의
- 서비스 가이드
- 사용 사례
- 사용자 리뷰

### 4.7 알림
- 이메일 알림
- 카카오톡 알림

### 4.8 통계 & 대시보드
- 포스팅 조회수 통계
- 대시보드 (홈)

### 4.9 결제
- 무료 티어 제공
- 월간 / 연간 요금제
- 결제 기능
- 티어별 월간 생성 수 가이드

### 4.10 마이페이지
- 내 캐릭터 / 글 작성 룰 설정
- 프로필 관리
