## Phase 3 확정 — 추가 설치 필요 패키지

다음 패키지들은 Phase 5 구현 시 설치 승인 필요:

### next-auth (Auth.js v5)
- **요청일**: 2026-03-31
- **요청 사유**: 소셜 로그인 (Google/Kakao/Apple) + 세션 관리
- **사용 위치**: `lib/auth.ts`, `app/api/auth/`
- **대안 검토**: 직접 JWT 구현 → 보안 리스크, 유지보수 부담
- **번들 사이즈**: ~50KB (서버 전용)
- **라이선스**: ISC
- **상태**: ⏳ 대기

### @prisma/client + prisma
- **요청일**: 2026-03-31
- **요청 사유**: PostgreSQL ORM, DB 스키마 단일 출처
- **사용 위치**: `prisma/schema.prisma`, `lib/prisma.ts`
- **대안 검토**: Drizzle → 학습 비용, 팀 선호도 낮음; 직접 pg 쿼리 → 타입 안전성 없음
- **번들 사이즈**: 서버 전용 (클라이언트 노출 없음)
- **라이선스**: Apache 2.0
- **상태**: ⏳ 대기

### zustand
- **요청일**: 2026-03-31
- **요청 사유**: 클라이언트 전역 상태 (사용자 정보, UI 상태)
- **사용 위치**: `stores/`
- **대안 검토**: Context API → 리렌더 성능, prop drilling 문제
- **번들 사이즈**: ~1KB
- **라이선스**: MIT
- **상태**: ⏳ 대기

- **요청일**: 2026-03-31

- **대안 검토**: SWR → tech-stack.md에서 금지 (통일성)
- **번들 사이즈**: ~13KB gzip
- **라이선스**: MIT
